Overview

What This Template Delivers

This is a complete implementation template for building a custom Agentforce agent that brings AI-powered marketing intelligence directly into your Salesforce org. It enables marketers to explore data, launch campaigns, and analyze performance through natural language.

The template is designed to be org-agnostic: it does not assume specific object names, table structures, or org configurations. Every component adapts at runtime through dynamic schema discovery and an AI grounding layer that translates raw metadata into business-meaningful context.

AI Grounding

What Makes It Accurate

A three-layer architecture ensures the agent understands your data in business terms, not just raw API names.

1

Einstein Data Prism

Automatic

Auto-scans all Data Cloud metadata — tables, fields, relationships, and sample values — then generates business-language descriptions stored in a vector database. At query time, returns semantically matched tables and fields.

2

Metadata Studio

Admin-Curated

An admin-facing UI for reviewing and refining auto-generated descriptions. Domain experts add business context that Data Prism alone can't infer — like "CPC = Cost per click in platform currency (USD)."

3

MI Semantic Data Model

Platform-Native

A pre-built semantic layer created automatically by the MI Data Pipeline. Defines metrics (CPC, CTR, ROAS), dimensions, and calculations that Tableau Semantics and the agent use for accurate analytics.

Core Engine

NL-to-SQL: 3-Layer Translation

How the agent translates "show me Meta Ads CPC by campaign" into a valid, accurate SQL query.

1

Runtime Schema Discovery

Before any query, the agent discovers what data exists on the org. The enhanced service returns tables, fields, types, record counts, and business descriptions from Data Prism — giving the LLM rich semantic context.

2

Schema-Aware Prompt Template

The discovered schema (with descriptions) is injected into a prompt template with SQL rules specific to Data Cloud — including ANSI SQL support, JOINs, window functions, and NULLIF() for safe division.

3

Validated Query Execution

The generated SQL is executed against Data Cloud's Query API. On error, the ReAct planner recovers by passing the error back to the prompt for correction — an automatic retry loop.

8-Phase Deployment

From foundation setup to demo-ready agent in ~2.5–3.5 hours.

Phase 1 ~45 min Manual + Automated

Marketing Intelligence Foundation

EMI Mock connectors, data pipeline, Einstein enrichment, campaign patterns, Tableau Next visualization, and MI goals.

Phase 1.5 — NEW ~15 min Manual

AI Grounding Layer

Provision Einstein Data Prism, review descriptions in Metadata Studio, verify the Marketing Intelligence Semantic Data Model.

Phase 2A Design Architecture

NL-to-SQL Translation Architecture

3-layer architecture design: runtime schema discovery, schema-aware prompt template, validated query execution with ReAct recovery.

Phase 2 ~45 min Automated

Apex Services & Prompt Templates

6 Apex services (schema, data explorer, CRM query, campaign, broadcast flow, performance) + 2 prompt templates (NL-to-SQL, report generation).

Phase 3 ~20 min Automated

GenAI Functions

8 functions mapping prompts and services to Agentforce: schema discovery, SQL generation, DMO query, CRM query, segment creation, broadcast flow trigger, performance analysis, report generation.

Phase 4 ~10 min Automated

GenAI Plugins

3 plugins: Data Explorer, Campaign & Journey Ops, Performance Analyst — each with detailed orchestration instructions.

Phase 5 ~15 min Automated

Planner, Bot & Permissions

ReAct planner with variable mappings, Marketing Copilot bot configuration, and permission set for agent access.

Phase 6 ~10 min Automated

Deployment Sequence

Strict dependency-order deployment of all 24 components: Apex → Prompt Templates → Functions → Plugins → Planner → Bot → Permission Set.

Phase 7 ~8 min Manual

Default Paid Media Agent

Enable Agentforce default agent and add Paid Media Topic from the asset library for campaign optimization.

Phase 8 ~20 min Manual

Validation, Quality & Demo Script

Full validation checklist, metadata quality review, and a three-act demo script: Discovery → Insight → Action.
