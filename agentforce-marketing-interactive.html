AI Grounding Layer

What makes it
accurate.

Three layers ensure the agent understands your data in business terms — not raw API names.

1

Einstein Data Prism

Automatic

Auto-scans metadata, generates descriptions, stores in vector DB. Returns semantically matched fields at query time.

2

Metadata Studio

Admin-Curated

Admins review and refine descriptions. Add business context like "CPC = Cost per click in USD."

3

MI Semantic Model

Platform-Native

Pre-built by MI Data Pipeline. Defines CPC, CTR, ROAS as proper metrics with calculation formulas.

Core Engine

NL-to-SQL: 3-Layer Translation

How "show me Meta Ads CPC by campaign" becomes a valid, accurate SQL query.

1

Schema Discovery

Discovers all available tables, fields, types, and enriched business descriptions from Data Prism at runtime.

↓
2

Prompt Template

Schema + descriptions injected into a prompt with Data Cloud SQL rules: ANSI SQL, JOINs, NULLIF() for safe division.

↓
3

Validated Execution

SQL executed against Data Cloud Query API. On error, ReAct planner retries with error context for automatic correction.
